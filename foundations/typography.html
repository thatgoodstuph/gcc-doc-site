Typography

Type scale, font families, and typography guidelines for consistent text styling across the system.

Type Scale

The typography system uses a consistent scale for font sizes, weights, and line heights.

Size Font Size Line Height Usage
3XL 32px 1.2 Hero headings, large displays
2XL 24px 1.3 Page titles, section headings
XL 20px 1.4 Subsection headings
LG 18px 1.5 Large body text, emphasized content
Base 16px 1.6 Body text, default size
SM 14px 1.5 Secondary text, captions
XS 12px 1.4 Labels, fine print

Font Families

The system uses a system font stack for optimal performance and consistency:

font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;

Font Weights

  • Regular (400) - Default body text
  • Medium (500) - Emphasized text, active states
  • Semibold (600) - Headings, important labels
  • Bold (700) - Strong emphasis, primary headings

Usage Guidelines

Do

  • Use the type scale consistently
  • Maintain proper line heights for readability
  • Use semantic HTML elements (h1, h2, p, etc.)

Don't

  • Hard-code font sizes in components
  • Use more than 3-4 font sizes on a single page
  • Create custom font weights outside the defined scale
